# 🧩 BashOps Toolkit

Набор Bash-скриптов для диагностики Linux-систем, проверки состояния и автоматизации рутинных задач DevOps.

Проект создаётся как практический инструмент — с упором на реальные кейсы, а не теорию.

---

## ⚡ Возможности

- Проверка состояния системы (CPU, RAM, диск, uptime)
- Анализ процессов и нагрузки
- Мониторинг сервисов (systemd)
- Работа с сетью и портами
- Анализ логов (включая ошибки 4xx/5xx)
- Модульная архитектура для сборки своих скриптов
- Переиспользуемые Bash-модули

---

## 🏗️ Философия проекта

Проект строится вокруг простой идеи:

## 📁 Структура проекта

- docs/ — документация для всех скриптов (описание, флаги, аргументы)
- src/modules/ — небольшие переиспользуемые модули (одна задача — один модуль)
- src/builds/ — скрипты, собранные из модулей
- src/ — скрипты более высокого уровня с расширенной функциональностью

---

## 📂 Общая структура

```
docs/
  builds/
    bash_1
    bash_2
  modules/
    bash_module_1.md
    bash_module_2.md
  bash_1.md
  bash_2.md

src/
  builds/
    bash_1
    bash_2
  modules/
    bash_module_1
    bash_module_2
  bash_1
  bash_2
```
